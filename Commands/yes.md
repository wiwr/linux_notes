`yes` - will print y
```bash
yes
```
yes can be used to print no
```bash
yes n
```
`yes "Something to print"`
```bash
yes "Something to print"
```
to confirm
```bash
yes | sudo apt install vim
```
```bash
yes | sudo apt install --auto-remove vim
```

to send 10 yes to output
```bash
yes | head -n 5
```
